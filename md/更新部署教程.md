# 洛克王国异色精灵展示系统 - 更新部署教程

## 一、更新流程（3种方式）

### 方式1：GitHub 流程（推荐）

```
本地修改 → GitHub → ECS 拉取 → 构建 → 完成
```

**本地操作：**

```bash
# 1. 进入项目目录
cd D:\Trae_project\开发1\洛克王国异色html\yise_Window

# 2. 添加修改的文件
git add .

# 3. 提交
git commit -m "更新说明"

# 4. 推送到 GitHub
git push
```

**ECS 操作（Workbench 终端）：**

```bash
# 5. 进入项目目录
cd /opt/luokewangguo-quanzhan

# 6. 拉取最新代码
git pull

# 7. 重新构建前端
cd frontend
npm run build

# 8. 重启后端
cd /opt/luokewangguo-quanzhan/backend
pkill -f "python3 main.py"
nohup python3 main.py > app.log 2>&1 &
```

**完成！** 刷新网页查看效果。

---

### 方式2：直接上传（不改 GitHub，快速测试）

适合：小改动、临时测试、不想提交 GitHub

```
本地修改 → 用 FileZilla 上传 → ECS 构建 → 完成
```

**本地操作：**
1. 修改代码
2. 打开 FileZilla，连接 ECS
3. 把修改的文件直接拖到 ECS 的对应位置

**ECS 操作（Workbench 终端）：**

```bash
cd /opt/luokewangguo-quanzhan/frontend
npm run build
cd /opt/luokewangguo-quanzhan/backend
pkill -f "python3 main.py"
nohup python3 main.py > app.log 2>&1 &
```

---

### 方式3：本地修改，本地看效果（开发调试）

适合：正在改代码、想实时看效果

```
本地修改 → 自动热更新 → 浏览器直接看
```

- 先启动本地服务
- 修改代码后，浏览器自动刷新
- 改满意了再按方式1部署到线上
