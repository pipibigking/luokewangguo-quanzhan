# 洛克王国异色宠物图鉴项目 - 文件架构文档

## 项目概述

本项目是一个基于 **Vue 3 + TypeScript + FastAPI** 的洛克王国异色宠物图鉴管理系统，用于展示和管理洛克王国游戏中的异色宠物信息。

## 1. 后端服务 (`backend/`)

### 目录结构

```
backend/
├── main.py              # 主应用入口
├── init_db.py           # 数据库初始化
├── migrate_db.py        # 数据库迁移
├── migrate_tables.py    # 表结构迁移
├── migrate_abilities.py # 技能数据迁移
├── requirements.txt     # Python 依赖列表
└── railway.json         # Railway 部署配置
```

### 文件说明

| 文件 | 作用 | 技术要点 |
|------|------|----------|
| `main.py` | FastAPI 主服务入口，定义 API 路由 | 包含宠物、公告、用户等 CRUD 接口 |
| `init_db.py` | 初始化数据库连接和基础配置 | 数据库连接池配置 |
| `migrate_db.py` | 数据库数据迁移脚本 | 批量导入宠物数据 |
| `migrate_tables.py` | 数据库表结构迁移 | 创建/更新表定义 |
| `migrate_abilities.py` | 宠物技能数据迁移 | 导入技能相关数据 |
| `requirements.txt` | Python 依赖声明 | FastAPI、SQLAlchemy、Uvicorn 等 |
| `railway.json` | Railway 平台部署配置 | 云服务部署参数 |

### 2.1 API 接口 (`src/api/`)

| 文件 | 作用 |
|------|------|
| `index.ts` | 封装所有后端 API 调用，包含宠物、公告、用户管理等接口 |

### 2.2 Vue 组件 (`src/components/`)

| 文件 | 作用 | 功能描述 |
|------|------|----------|
| `AdminLayout.vue` | 后台管理布局组件 | 侧边栏导航、顶部栏、主内容区域 |
| `AnnouncementBar.vue` | 公告栏组件 | 展示系统公告信息 |
| `FilterBar.vue` | 筛选栏组件 | 宠物属性、组别筛选功能 |
| `PetCard.vue` | 宠物卡片组件 | 展示单只宠物的缩略信息 |
| `PetModal.vue` | 宠物详情弹窗 | 展示宠物完整信息 |

### 2.3 页面组件 (`src/pages/`)

| 文件 | 作用 | 功能描述 |
|------|------|----------|
| `HomePage.vue` | 首页 | 宠物列表展示、筛选搜索 |
| `LoginPage.vue` | 登录页 | 用户登录、权限验证 |
| `PetManagePage.vue` | 宠物管理页 | 宠物数据增删改查 |
| `AnnouncementManagePage.vue` | 公告管理页 | 系统公告管理 |
| `AccountManagePage.vue` | 账号管理页 | 用户账号管理 |

### 2.4 工具函数 (`src/utils/`)

| 文件 | 作用 | 功能描述 |
|------|------|----------|
| `attributeIcons.ts` | 属性图标映射 | 宠物属性与图标文件的对应关系 |
| `auth.ts` | 认证工具函数 | Token 管理、权限验证 |
| `groupColors.ts` | 组别颜色映射 | 宠物组别与颜色的对应关系 |

### 2.5 配置文件

| 文件 | 作用 |
|------|------|
| `router/index.ts` | Vue Router 路由配置 | 定义页面路由和导航守卫 |
| `types/index.ts` | TypeScript 类型定义 | 宠物、用户、公告等数据类型 |

## 3. 静态图片资源 (`images/`)

### 目录结构

```
images/
├── attribute/           # 属性图标
├── balls/               # 捕捉球图标
└── pets/                # 宠物图片
```

### 文件说明

| 目录 | 内容 | 数量 |
|------|------|------|
| `attribute/` | 18种宠物属性图标（光系、冰系、地系等） | 18个 |
| `balls/` | 14种捕捉球图标（国王球、光合球等） | 14个 |
| `pets/` | 异色宠物图片（双灯鱼、大耳帽兜等） | 21个 |

## 项目运行流程

```
启动后端 → 启动前端 → 用户访问 → 登录验证 → 浏览宠物 → 管理操作
    ↓              ↓
  main.py      vite dev
```

## 关键功能模块

1. **宠物展示模块**：宠物列表、筛选、详情查看
2. **宠物管理模块**：宠物数据的增删改查
3. **公告管理模块**：系统公告发布与管理
4. **用户管理模块**：管理员账号管理
5. **认证授权模块**：登录验证、权限控制
