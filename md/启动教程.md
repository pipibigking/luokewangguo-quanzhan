# 洛克王国异色宠物图鉴 - 项目启动教程

## 项目简介

本项目是一个基于 **Vue 3 + TypeScript + FastAPI** 的洛克王国异色宠物图鉴管理系统，包含前端展示系统和后台管理系统。

---

## 环境要求

- Python 3.8+
- pip（Python 包管理器）

---

## 项目结构

```
yise_Window/
├── backend/              # 后端服务（FastAPI）
│   ├── main.py          # 主入口
│   ├── system_data.db   # SQLite 数据库
│   └── requirements.txt # Python 依赖
├── frontend/             # 前端应用（Vue 3）
│   ├── src/             # 源代码
│   └── package.json     # Node 依赖
├── images/              # 静态图片资源
├── start.bat           # Windows 一键启动脚本
└── deploy.bat          # 完整部署脚本
```

---

## 快速启动（推荐）

### 方法一：使用一键启动脚本（最简单）

1. 双击项目根目录下的 `start.bat` 文件
2. 等待服务启动完成
3. 打开浏览器访问：**http://localhost:8004**

### 方法二：使用 CMD 命令行启动

1. 打开命令提示符（CMD）或 PowerShell
2. 进入项目目录：
   ```cmd
   cd d:\Trae_project\开发1\洛克王国异色html\yise_Window
   ```
3. 一键启动（首次需要安装依赖）：
   ```cmd
   start.bat
   ```
4. 或者手动启动后端：
   ```cmd
   cd backend
   python main.py
   ```

## 详细启动步骤

### 第一步：安装依赖（仅首次）

1. 进入后端目录：
   ```cmd
   cd d:\Trae_project\开发1\洛克王国异色html\yise_Window\backend
   ```

2. 安装 Python 依赖：
   ```cmd
   pip install -r requirements.txt
   ```

### 第二步：启动服务

1. 在后端目录下运行：
   ```cmd
   python main.py
   ```

2. 看到以下提示表示启动成功：
   ```
   ==================================================
     洛克王国异色精灵展示系统
     API 文档: http://localhost:8004/docs
     前端页面: http://localhost:8004/
   ==================================================
   ```

## 常见问题

### 1. 后端启动报错 "端口已被占用"
```
Error: [Errno 10048] 通常每个套接字地址只能使用一次
```
**解决方法**：关闭占用 8004 端口的程序，或修改 `backend/main.py` 中的端口号。

### 2. 页面显示黑屏或加载失败
1. 确保访问的是 **http://localhost:8004/**（不是 5173）
2. 检查后端服务是否正常运行
3. 刷新浏览器页面

### 3. 图片无法显示
1. 确认 `images/` 目录位于项目根目录
2. 确认 `backend/main.py` 中图片目录路径正确
3. 检查浏览器开发者工具 Network 面板

### 4. 登录失败
1. 确认使用的是默认账号：`admin` / `admin123`
2. 首次启动时，后端会自动创建默认管理员账号
3. 检查浏览器控制台是否有报错信息

## 开发模式（前端开发用）

如果需要修改前端代码并实时预览：

1. 启动后端（保持运行）
2. 新开一个终端，进入前端目录：
   ```cmd
   cd frontend
   npm install  # 仅首次
   npm run dev
   ```
3. 访问 http://localhost:5173 进行开发
