# 洛克王国异色宠物图鉴 - 项目启动教程

## 项目简介

本项目是一个基于 **Vue 3 + TypeScript + FastAPI** 的洛克王国异色宠物图鉴管理系统，包含前端展示系统和后台管理系统。

---

## 环境要求

### 后端环境
- Python 3.8+
- pip（Python 包管理器）

### 前端环境
- Node.js 16+
- npm（Node.js 自带）

---

## 项目结构

```
yise_Window/
├── backend/              # 后端服务（FastAPI）
│   ├── main.py          # 主入口
│   ├── system_data.db   # SQLite 数据库
│   └── requirements.txt # Python 依赖
├── frontend/             # 前端应用（Vue 3）
│   ├── src/             # 源代码
│   └── package.json     # Node 依赖
└── images/              # 静态图片资源
```

## 启动步骤

### 第一步：启动后端服务

1. 进入后端目录：
   ```bash
   cd d:\Trae_project\开发1\洛克王国异色html\yise_Window\backend
   ```

2. 安装 Python 依赖：
   ```bash
   pip install -r requirements.txt
   ```
   依赖列表：
   - fastapi==0.104.1
   - uvicorn==0.24.0
   - sqlalchemy==2.0.31
   - python-dotenv==1.0.0

3. 启动后端服务：
   ```bash
   python main.py
   ```

4. 验证后端是否启动成功：
   - 访问 http://localhost:8004
   - 看到 `{"message":"洛克王国异色精灵展示 API","docs":"/docs","api":"/api/pets"}` 表示成功
   - API 文档地址：http://localhost:8004/docs

### 第二步：启动前端服务

1. 进入前端目录：
   ```bash
   cd d:\Trae_project\开发1\洛克王国异色html\yise_Window\frontend
   ```

2. 安装 Node 依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 访问前端页面：
   - 打开浏览器访问 http://localhost:5173（或其他提示的端口）
   - 默认端口为 5173

## 常见问题

### 1. 后端启动报错 "端口已被占用"
```
Error: [Errno 10048] 通常每个套接字地址只能使用一次
```
**解决方法**：关闭占用 8004 端口的程序，或修改 `backend/main.py` 中的端口号。

### 2. 前端无法连接后端 API
检查以下几点：
1. 后端服务是否已启动（确认 http://localhost:8004 可访问）
2. 前端代理配置是否正确（参考 `frontend/vite.config.ts`）
3. 浏览器控制台是否有 CORS 错误

### 3. 图片无法显示
1. 确认 `images/` 目录位于项目根目录
2. 确认 `backend/main.py` 中图片目录路径正确
3. 检查浏览器开发者工具 Network 面板

### 4. 登录失败
1. 确认使用的是默认账号：`admin` / `admin123`
2. 首次启动时，后端会自动创建默认管理员账号
3. 检查浏览器控制台是否有报错信息

## 生产环境构建

### 前端构建
```bash
cd frontend
npm run build
```
构建产物位于 `frontend/dist/` 目录，可部署到 Vercel 等平台。

### 后端部署
后端默认使用 SQLite 数据库，可直接部署到 Railway、Heroku 等平台。
