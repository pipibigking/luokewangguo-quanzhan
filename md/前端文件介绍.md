# 前端文件结构详解

## 项目概述

这是一个基于 **Vue 3 + TypeScript + Vite + Tailwind CSS** 构建的洛克王国异色精灵展示与管理系统的前端项目。

---

## 目录结构

```
frontend/
├── dist/                          # 构建输出目录
│   ├── index.html                # 构建后的入口文件
│   └── assets/                   # 打包后的资源文件
│       ├── *.js                  # JS 代码
│       └── *.css                 # CSS 样式
├── src/                          # 源代码目录
│   ├── api/                      # API 请求模块
│   │   └── index.ts             # 所有 API 请求封装
│   ├── components/               # 可复用组件
│   │   ├── AdminLayout.vue      # 后台管理布局组件
│   │   ├── PetCard.vue          # 精灵卡片组件
│   │   ├── PetModal.vue         # 精灵详情弹窗
│   │   ├── FilterBar.vue        # 筛选栏组件
│   │   └── AnnouncementBar.vue  # 公告栏组件
│   ├── pages/                    # 页面组件
│   │   ├── HomePage.vue         # 首页 - 精灵展示
│   │   ├── LoginPage.vue        # 登录页
│   │   ├── PetManagePage.vue    # 精灵管理页
│   │   ├── AnnouncementManagePage.vue  # 公告管理页
│   │   └── AccountManagePage.vue       # 账号管理页
│   ├── router/                   # 路由配置
│   │   └── index.ts             # 路由表和路由守卫
│   ├── types/                    # TypeScript 类型定义
│   │   └── index.ts             # 类型声明
│   ├── utils/                    # 工具函数
│   │   ├── auth.ts              # 认证相关工具
│   │   ├── attributeIcons.ts    # 属性图标映射
│   │   └── groupColors.ts       # 分组颜色映射

│   ├── main.ts                   # 应用入口
│   └── style.css                 # 全局样式
├── index.html                    # HTML 入口文件
├── package.json                  # 项目依赖和脚本
├── package-lock.json             # npm 依赖锁定

├── tsconfig.node.json            # TypeScript Node 配置
├── vite.config.ts                # Vite 配置
└── tailwind.config.js            # Tailwind CSS 配置
```

## 文件详细说明

### 根目录文件

#### 1. `index.html`
- **作用**：HTML 入口文件
- **内容**：
  - 页面元数据配置
  - 引入 Google Fonts（ZCOOL KuaiLe、Ma Shan Zheng、Noto Sans SC 等）
  - 应用挂载点 `<div id="app"></div>`

#### 2. `package.json`
- **作用**：项目配置和依赖管理
- **主要内容**：
  ```json
  {
    "name": "locke-pets-frontend",
    "version": "1.0.0",
    "scripts": {
      "dev": "vite",           // 启动开发服务器
      "build": "vue-tsc && vite build",  // 构建生产版本
      "preview": "vite preview" // 预览构建结果
    },
    "dependencies": {
      "vue": "^3.3.13",        // Vue 3 框架
      "vue-router": "^4.6.4",  // 路由
      "axios": "^1.6.2",       // HTTP 客户端
      "animejs": "^3.2.2"      // 动画库
    }
  }
  ```

#### 3. `vite.config.ts`
- **作用**：Vite 构建工具配置
- **主要配置**：
  - 路径别名：`@` 指向 `src` 目录
  - 开发服务器代理：
    - `/api` → `http://localhost:8004`
    - `/images` → `http://localhost:8004`
    - `/icons` → `http://localhost:8004`

#### 4. `tailwind.config.js`
- **作用**：Tailwind CSS 主题配置
- **扩展配置**：
  - **颜色**：主题配色（金色、橙色、红色等）
  - **字体**：游戏风格字体
  - **阴影**：魔法效果阴影

---

### `src/` 源代码目录

#### 1. `main.ts`
- **作用**：应用入口文件
- **功能**：
  - 创建 Vue 应用实例
  - 注册路由
  - 挂载到 `#app`
  - 导入全局样式

#### 2. `App.vue`
- **作用**：根组件
- **功能**：
  - 路由视图容器 `<router-view />`
  - 全局样式定义（背景渐变、字体等）

#### 3. `style.css`
- **作用**：全局样式文件

---

### `src/pages/` 页面组件

| 文件 | 路由 | 功能描述 |
|------|------|----------|
| `HomePage.vue` | `/` | 前台精灵展示页面，包含筛选、搜索、排序功能 |
| `LoginPage.vue` | `/login` | 管理员登录页面 |
| `PetManagePage.vue` | `/admin/pets` | 精灵管理（增删改查、价格内联编辑） |
| `AnnouncementManagePage.vue` | `/admin/announcement` | 公告管理 |
| `AccountManagePage.vue` | `/admin/accounts` | 管理员账号管理 |

---

### `src/components/` 可复用组件

| 组件 | 功能 |
|------|------|
| `AdminLayout.vue` | 后台管理布局（侧边栏导航、主题色为紫蓝渐变） |
| `PetCard.vue` | 精灵卡片（前台展示用，包含图片、名称、属性、价格等） |
| `PetModal.vue` | 精灵详情弹窗 |
| `FilterBar.vue` | 筛选栏组件（分组、搜索、排序） |
| `AnnouncementBar.vue` | 公告栏组件 |

---

### `src/router/` 路由配置

#### `index.ts`
- **路由模式**：Hash 模式
- **路由表**：
  - `/` → 首页（HomePage）
  - `/login` → 登录页（LoginPage）
  - `/admin` → 后台管理（嵌套路由）
    - `/admin/pets` → 精灵管理
    - `/admin/announcement` → 公告管理
    - `/admin/accounts` → 账号管理
- **路由守卫**：访问 `/admin` 开头路径时检查登录状态，未登录则跳转到登录页

---

### `src/api/` API 请求模块

#### `index.ts`
集中管理所有 API 请求，封装成独立函数

| 分类 | 函数 |
|------|------|
| **精灵管理** | `getPets()`, `getPetById()`, `createPet()`, `updatePet()`, `deletePet()`, `togglePetActive()`, `batchActivatePets()`, `batchDeactivatePets()` |
| **分组管理** | `getGroups()`, `getGroupColors()`, `updateGroupColor()` |
| **公告管理** | `getAnnouncement()`, `updateAnnouncement()` |
| **账号管理** | `getAdminAccounts()`, `createAdminAccount()`, `updateAdminAccount()`, `deleteAdminAccount()`, `adminLogin()` |

---

### `src/utils/` 工具函数

#### 1. `auth.ts`
认证相关工具函数
- `login(username, password)` - 登录
- `logout()` - 登出
- `isAuthenticated()` - 检查是否已认证
- `getCurrentAdmin()` - 获取当前登录用户
- `addAccount()`, `updateAccount()`, `deleteAccount()`, `getAccounts()` - 账号管理

#### 2. `attributeIcons.ts`
属性图标路径映射

#### 3. `groupColors.ts`
分组颜色映射

---

### `src/types/` 类型定义

#### `index.ts`
TypeScript 接口定义

```typescript
export interface Pet {
  id: number
  name: string
  group: string
  image_url: string
  price: number
  attributes: string[]
  description: string
  abilities: string[]
  is_active: boolean
}

export interface FilterOptions {
  group: string | null
  search: string
  sortBy: string
  sortOrder: 'asc' | 'desc'
  attribute: string | null
}

export interface Announcement {
  id: number
  content: string
  updated_at: string
}
```

## 开发说明

### 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

### 环境要求

- Node.js 16+
- 后端服务需运行在 `http://localhost:8004`

## 项目特点

1. **模块化架构**：页面、组件、API、工具分离，便于维护
2. **类型安全**：使用 TypeScript 提供完整类型支持
3. **现代化构建**：Vite 快速构建，热更新
4. **响应式设计**：Tailwind CSS 构建美观界面
5. **路由守卫**：后台路由权限控制
6. **集中式 API 管理**：所有请求集中在 `api/index.ts`
