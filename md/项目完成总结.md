## 快速启动指南

### 1. 安装依赖

**后端**:
```bash
cd backend
pip install -r requirements.txt
```

**前端**:
```bash
cd frontend
npm install
```

### 2. 初始化数据库

```bash
cd backend
python init_db.py
```

### 3. 启动服务

**后端** (终端1):
```bash
cd backend
python main.py
```
后端运行在: `http://localhost:8000`

**前端** (终端2):
```bash
cd frontend
npm run dev
```
前端运行在: `http://localhost:5173`

### 4. 生成属性图标

```bash
python scrape_attributes.py
```
然后打开 `attribute_icons/preview.html` 预览

## API接口

### 获取精灵列表
```

```

```
GET /api/pets/{id}
```

### 获取所有分组
```
GET /api/groups
```

## 故障排查

### 图片不显示
1. 检查后端是否在运行
2. 直接访问图片URL测试
3. 查看浏览器控制台错误
4. 运行 `diagnose.py` 检查问题

### 数据库错误
1. 确认已运行 `init_db.py`
2. 检查 `locke_pets.db` 是否存在
3. 查看后端日志

### 前端无法启动
1. 确认已运行 `npm install`
2. 检查 Node.js 版本
3. 使用 CMD 而不是 PowerShell
