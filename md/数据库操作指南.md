## 一、数据库概述

### 1.1 数据库配置
- **数据库类型**: SQLite
- **数据库文件位置**: `backend/locke_pets.db`
- **配置文件**: `backend/main.py`

### 1.2 数据模型定义

```python
class Pet(Base):
    __tablename__ = "pets"
    
    id = Column(Integer, primary_key=True, autoincrement=True)  # 精灵ID
    name = Column(String(50), nullable=False)                     # 精灵名称
    group = Column(String(50), nullable=False)                    # 分组
    image_url = Column(String(255), nullable=False)               # 图片路径
    price = Column(Integer, nullable=False)                       # 价格
    attributes = Column(Text, default="[]")                       # 属性 (JSON数组)
    description = Column(Text, default="")                        # 描述
    created_at = Column(DateTime, default=func.now())             # 创建时间
```

## 四、增删改操作

### 4.1 添加新精灵
```python
def add_pet(name, group, image_url, price, attributes=None, description=""):
    db = SessionLocal()
    new_pet = Pet(
        name=name,
        group=group,
        image_url=image_url,
        price=price,
        attributes=json.dumps(attributes if attributes else []),
        description=description
    )
    db.add(new_pet)
    db.commit()
    db.refresh(new_pet)
    pet_id = new_pet.id
    db.close()
    return pet_id

# 使用示例
add_pet(
    name="新精灵",
    group="活动异色",
    image_url="images/new_pet.png",
    price=80,
    attributes=["火"],
    description="这是一个新的精灵"
)
```

### 4.2 更新精灵信息
```python
def update_pet(pet_id, **kwargs):
    db = SessionLocal()
    pet = db.query(Pet).filter(Pet.id == pet_id).first()
    if not pet:
        db.close()
        return False
    
    for key, value in kwargs.items():
        if key == "attributes":
            pet.attributes = json.dumps(value)
        elif hasattr(pet, key):
            setattr(pet, key, value)
    
    db.commit()
    db.close()
    return True

# 使用示例
update_pet(1, name="改名后的精灵", price=90)
```

### 4.3 删除精灵
```python
def delete_pet(pet_id):
    db = SessionLocal()
    pet = db.query(Pet).filter(Pet.id == pet_id).first()
    if pet:
        db.delete(pet)
        db.commit()
        db.close()
        return True
    db.close()
    return False

# 使用示例
delete_pet(19)
```

## 六、数据结构说明

### 6.1 精灵数据结构
```json
{
  "id": 1,
  "name": "大耳帽兜",
  "group": "赛季异色",
  "image_url": "images/大耳帽兜.png",
  "price": 60,
  "attributes": ["冰"],
  "description": "大耳帽兜是一只可爱的冰系精灵...",
  "created_at": "2024-05-11T12:00:00"
}
```

### 6.2 属性说明
- **id**: 唯一标识符，整数
- **name**: 精灵名称，最长50字符
- **group**: 分组类型，包含：
  - 赛季异色
  - 赛季奇遇异色
  - 赛季战令异色
  - 活动异色
- **image_url**: 图片相对路径
- **price**: 价格，整数
- **attributes**: 属性数组，JSON格式，支持的属性：
  - 普、草、火、水、光、地、冰、龙、电、毒、虫、武、翼、萌、幽、恶、机、幻
- **description**: 精灵描述，文本格式
- **created_at**: 创建时间，自动生成

## 七、数据库初始化

### 7.1 手动初始化数据库
```bash
cd backend
```

### 7.2 初始化脚本内容
```python
# init_db.py
from main import Base, engine, SessionLocal
import json

# 精灵数据列表
pets_data = [
    {
        "id": 1,
        "name": "大耳帽兜",
        "group": "赛季异色",
        "image_url": "images/大耳帽兜.png",
        "price": 60,
        "attributes": ["冰"],
        "description": "..."
    },
    # ... 更多精灵
]

def init_db():
    Base.metadata.create_all(bind=engine)
    db = SessionLocal()
    
    from main import Pet
    
    for pet_data in pets_data:
        existing_pet = db.query(Pet).filter(Pet.id == pet_data["id"]).first()
        if existing_pet:
            # 更新已有精灵
            existing_pet.name = pet_data["name"]
            existing_pet.group = pet_data["group"]
            existing_pet.image_url = pet_data["image_url"]
            existing_pet.price = pet_data["price"]
            existing_pet.attributes = json.dumps(pet_data["attributes"])
            existing_pet.description = pet_data["description"]
        else:
            # 添加新精灵
            pet = Pet(
                name=pet_data["name"],
                group=pet_data["group"],
                image_url=pet_data["image_url"],
                price=pet_data["price"],
                attributes=json.dumps(pet_data["attributes"]),
                description=pet_data["description"]
            )
            db.add(pet)
    
    db.commit()
    db.close()
    print("数据库初始化完成！")

if __name__ == "__main__":
    init_db()
```

## 八、故障排查

### 8.1 数据库不存在
- **问题**: `SQLite.OperationalError: no such table: pets`
- **解决**: 运行 `python init_db.py` 创建表

### 8.2 属性解析失败
- **问题**: `json.JSONDecodeError`
- **解决**: 确保 attributes 字段存储的是有效的JSON格式

### 8.3 数据库锁定
- **问题**: `database is locked`
- **解决**: 确保没有其他进程正在访问数据库，重启服务

```python
import json

def export_to_json(filename="pets_export.json"):
    db = SessionLocal()
    pets = db.query(Pet).all()
    export_data = []
    for pet in pets:
        export_data.append({
            "id": pet.id,
            "name": pet.name,
            "group": pet.group,
            "image_url": pet.image_url,
            "price": pet.price,
            "attributes": json.loads(pet.attributes),

        json.dump(export_data, f, ensure_ascii=False, indent=2)
    db.close()
    print(f"数据已导出到 {filename}")
```
