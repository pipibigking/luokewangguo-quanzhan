### 1.2 数据模型定义

#### 1.2.1 精灵表 (pets)

```python
class Pet(Base):
    __tablename__ = "pets"
    
    id = Column(Integer, primary_key=True, autoincrement=True)  # 精灵ID
    name = Column(String(50), nullable=False)                     # 精灵名称
    group = Column(String(50), nullable=False)                    # 分组
    image_url = Column(String(255), nullable=False)               # 图片路径
    price = Column(Integer, nullable=False)                       # 价格
    attributes = Column(Text, default="[]")                       # 属性 (JSON数组)
    description = Column(Text, default="")                        # 描述
    created_at = Column(DateTime, default=func.now())             # 创建时间
    is_active = Column(Boolean, default=True)                     # 是否启用
```

#### 1.2.2 管理员账号表 (admin_accounts)

```python
class AdminAccount(Base):
    __tablename__ = "admin_accounts"
    
    id = Column(Integer, primary_key=True, autoincrement=True)    # 账号ID
    username = Column(String(50), unique=True, nullable=False)    # 用户名
    password = Column(String(255), nullable=False)                # 密码(加密)
    created_at = Column(DateTime, default=func.now())             # 创建时间
```

#### 1.2.3 公告表 (announcement)

```python
class Announcement(Base):
    __tablename__ = "announcement"
    
    id = Column(Integer, primary_key=True, autoincrement=True)    # 公告ID
    content = Column(Text, nullable=False)                        # 公告内容
    updated_at = Column(DateTime, default=func.now())             # 更新时间
```

## 四、增删改操作

### 4.1 添加新精灵
```python
def add_pet(name, group, image_url, price, attributes=None, description=""):
    db = SessionLocal()
    new_pet = Pet(
        name=name,
        group=group,
        image_url=image_url,
        price=price,
        attributes=json.dumps(attributes if attributes else []),
        description=description,
        is_active=True
    )
    db.add(new_pet)
    db.commit()
    db.refresh(new_pet)
    pet_id = new_pet.id
    db.close()
    return pet_id
```

### 4.2 更新精灵信息
```python
def update_pet(pet_id, **kwargs):
    db = SessionLocal()
    pet = db.query(Pet).filter(Pet.id == pet_id).first()
    if not pet:
        db.close()
        return False
    
    for key, value in kwargs.items():
        if key == "attributes":
            pet.attributes = json.dumps(value)
        elif hasattr(pet, key):
            setattr(pet, key, value)
    
    db.commit()
    db.close()
    return True

# 使用示例
update_pet(1, name="改名后的精灵", price=90)
```

### 6.3 属性说明

**精灵表字段**:
- **id**: 唯一标识符，整数
- **name**: 精灵名称，最长50字符
- **group**: 分组类型，包含：
  - 赛季异色
  - 赛季奇遇异色
  - 赛季战令异色
  - 活动异色
- **image_url**: 图片相对路径
- **price**: 价格，整数
- **attributes**: 属性数组，JSON格式，支持的属性：
  - 普、草、火、水、光、地、冰、龙、电、毒、虫、武、翼、萌、幽、恶、机、幻
- **description**: 精灵描述，文本格式
- **is_active**: 是否启用，布尔值
- **created_at**: 创建时间，自动生成

## 八、故障排查

### 8.1 数据库不存在
- **问题**: `SQLite.OperationalError: no such table: pets`
- **解决**: 重启后端服务，表会自动创建

### 8.2 属性解析失败
- **问题**: `json.JSONDecodeError`
- **解决**: 确保 attributes 字段存储的是有效的JSON格式

### 8.3 数据库锁定
- **问题**: `database is locked`
- **解决**: 确保没有其他进程正在访问数据库，重启服务

### 8.4 图片加载失败
- **问题**: 精灵图片显示为空白或404
- **解决**: 检查 image_url 是否正确，确保图片文件存在于 `images/pets/` 目录下

```python
def import_from_json(filename="pets_export.json"):
    import json
    with open(filename, "r", encoding="utf-8") as f:
        import_data = json.load(f)
    
    db = SessionLocal()
    from main import Pet
    
    for pet_data in import_data:
        existing_pet = db.query(Pet).filter(Pet.id == pet_data["id"]).first()
        if existing_pet:
            existing_pet.name = pet_data["name"]
            existing_pet.group = pet_data["group"]
            existing_pet.image_url = pet_data["image_url"]
            existing_pet.price = pet_data["price"]
            existing_pet.attributes = json.dumps(pet_data["attributes"])
            existing_pet.description = pet_data["description"]
            existing_pet.is_active = pet_data.get("is_active", True)
        else:
            pet = Pet(
                id=pet_data["id"],
                name=pet_data["name"],
                group=pet_data["group"],
                image_url=pet_data["image_url"],
                price=pet_data["price"],
                attributes=json.dumps(pet_data["attributes"]),
                description=pet_data["description"],
                is_active=pet_data.get("is_active", True)
            )
            db.add(pet)
```
