## 常见问题

### Q: 图片无法显示？
A: 确保：
1. 后端正在运行（端口 8000）
2. `images` 文件夹存在且包含图片
3. 检查浏览器控制台的 Network 标签

### Q: 属性图标无法显示？
A: 确保：
1. `wiki_style_icons` 文件夹存在
2. 后端已挂载 `/icons` 路径

### Q: 如何重置数据库？
A: 运行：
```cmd
cd backend
python init_db.py
```
